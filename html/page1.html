Difference between programmer and designer


- A programmer probably works on the logic of the application. They make the buttons work when clicked, insert data into the database and retrieve that data. They make the application have a purpose.

-A designer is there to make the application look beautiful to the user, but more importantly help make it more user friendly so the final users understand what they have to do.

-You can have a programmer without a designer but the best combination is to have experts in both fields to create the best possible product.
